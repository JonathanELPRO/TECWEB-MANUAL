Selectores
Selector Descripcion Ejemplo de uso
Universal 
                        Selecciona todos los elementos. 
                        Sintaxis 
                        *{ declaraciones CSS }
                    
                        *{
                            color: green;
                        } 
                        Seleccionamos todos los elementos del documento.
Tipo 
                        Permite seleccionar elementos usando el nombre del elemento.
                        Sintaxis
                        nombre_elemento { declaraciones CSS }
                    
                        p{
	                        color: red;
                        } 
                        Seleccionamos todos los elementos p del documento.
De clase 
                        Permite seleccionar elementos en función de los valores del atributo class de los elementos.  
                        Para que los elementos sean seleccionados, su atributo class debe incluir un valor  que coincida con el valor del selector. 
                        Sintaxis 
                        .valor_atributo { declaraciones CSS }

                    
                        <p class=”foo bar”> ¡HOLA MUNDO! <p/p>
                        <p class=”foo baz”> ¡ADIÓS MUNDO! <p/p>

                        .foo{
                            color: blue;
                        }
                        Seleccionamos todos los elementos cuyo atributo class tenga como uno de sus valores la cadena foo.
De ID 
                        Permite seleccionar un elemento en función del valor del atributo ID del elemento. Para que el elemento sea seleccionado, su atributo id debe coincidir exactamente con el valor dado en el selector.
                        Sintaxis
                        #valor_atributo { declaraciones CSS }
                    
                        <p id=”foo”> ¡HOLA MUNDO! </p>

                        #foo{
                            color: blue;
                        }

                        Seleccionamos el elemento que tenga el atributo id con el valor foo.

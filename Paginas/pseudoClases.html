Permite seleccionar elementos en base a su estado (:disabled), posicion de los elementos en el documento(:first-child) e interacción del usuario(:hover) esta es la sintaxis :selector:pseudo-class { … }

Selectores
Selector Descripcion Ejemplo de uso
button:hover {...} 
                        La pseudo clase hover nos permite jugar con el estilo que tendra el elemento <button> cuando el usuario ponga el cursor sobre el.
                    
                        button:hover {
                            color: blue;
                          }
input:disabled{...} 
                        La pseudo clase disabled nos ayudar a manejar el estilo de aquellos elementos que hallan implementado su estado en disabled.
                        Podemos tener muchos inputs pero la regla o estilo que se mostrara en el ejemplo solo afectara a aquellos que hallan puesto su estado en disabled.
                    
                        HTML
                        <form>
                            <label for="fecha">Fecha</label>
                            <input name="fecha" type="date" disabled>
                        </form>
                        CSS                    
                        input:disabled{
                            background-color: dimgrey;
                            color: linen;
                        }
p:first-child{...} 
                        La pseudo clase first child hara que todo elemento <p> que son el primer hijo de algun otro elemento agarren estilo
                    
                        p:first-child{
                            color
                        }

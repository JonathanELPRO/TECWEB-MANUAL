tbody El elemento HTML body encapsula un conjunto de filas de tabla (elementos tr), lo que indica que comprende el cuerpo de la tabla (tabla).
  • Si la tabla incluye un bloque thead (para identificar semánticamente una fila de encabezados de columna) , el bloque tbody debe venir después de él.
  • Si los elementos tr se especifican fuera de un elemento tbody existente, como hijos directos del table , estos elementos serán encapsulados por un elemento tbody separado generado por el navegador.
  • Al imprimir un documento, los elementos thead y tfoot especifican información que puede lo mismo, o al menos muy similar, en cada página de una tabla de varias páginas, mientras que el tbody el contenido del elemento generalmente diferirá de una página a otra.
  • Cuando una tabla se presenta en un contexto de pantalla (como una ventana) que no es lo suficientemente grande para muestra toda la tabla, el agente de usuario puede permitir que el usuario desplace el contenido del thead , bloques tbody, tfoot y cantion separados entre sí para la misma tabla principal.
  • Puede utilizar más de un tbody por tabla, siempre que todos sean consecutivos. Esto le permite divida las filas en tablas grandes en secciones, cada una de las cuales puede formatearse por separado si es así deseable. Si no se marcan como elementos consecutivos, los navegadores corregirán este error de autor, asegurarse de que los elementos thead y tfoot se representen como el primer y el último elemento de la tabla, respectivamente.

caption El elemento HTML caption especifica el título (o título) de una tabla.

Si se utiliza, el elemento caption debe ser el primer elemento secundario de su elemento principal table.

Cuando el elemento table que contiene el caption es el único descendiente de un elemento figure, debe usar el elemento figcaption en lugar de caption.

Un color de fondo en la tabla no incluirá el título. Agregue un color de fondo al elemento caption también si desea que el mismo color esté detrás de ambos.
a

El elemento HTML a (o elemento de anclaje), con su atributo href, crea un hipervínculo a páginas web, archivos, direcciones de correo electrónico, ubicaciones en la misma página o cualquier otra cosa a la que pueda dirigirse una URL.

El contenido dentro de cada a debe indicar el destino del enlace. Si el atributo href está presente, al presionar la tecla enter mientras se enfoca en el elemento a se activará.

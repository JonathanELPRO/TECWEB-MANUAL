Selectores
Selector Descripcion Ejemplo de uso
selector[attr] { … } 
                        Selecciona elementos que tengan un atributo con el nombre attr.
                    
                        p[title]
                        {
                            background-color: green;
                        }
                        Seleccionamos todos los elementos <p> que tienen un atributo con nombre title.
selector[attr=value] { … } 
                        Selecciona elementos que tengan un atributo con el nombre attr. El único valor del atributo es exactamente value.
                    
                        p[title="parrafo2"]
                        {
                            background-color: yellow;
                        }
                        Lo que hace lo de arriba es a todos los elementos p con atributo de nombre title y valor igual a parrafo 2 les da estilo.
selector[attr~=value] { … } 
                        Selecciona los elementos que tengan un atributo con el nombre attr. Uno de los valores del atributo es exactamente value. 
                        Esto considerando que el valor de un atributo no necesariamente es unico pueden ser varios separados por espacios.
                    
                        div[class~=bar]
                        {
                            background-color: green;
                        } 
                        Selecciona los elementos que tengan un atributo con el nombre  class y al menos uno de los valores del atributo llamado class es bar .
selector[attr ^= value] { … } 
                        Selecciona los elementos que tengan un atributo con el nombre attr. El primero de los  valores del atributo empieza con value. 
                    
                        div[class^=ba]
                        {
                            background-color: red;
                        }
                        Selecciona los elementos que tengan un atributo con el nombre class y el primero de los  valores del atributo de nombre class es ba.
selector[attr $= value] { … } 
                        Selecciona los elementos que tengan un atributo con el nombre attr. El último de los  valores del atributo termina con value.
                    
                        div[class $= ar] {
                            background-color: red;
                        }
                        Selecciona los elementos que tengan un atributo con el nombre class. El último de los  valores del atributo llamado class es ar.
selector[attr *= value] { … } 
                        Selecciona los elementos que tengan un atributo con el nombre attr. Uno de los valores del atributo contiene al menos una ocurrencia de  value. 
                    
                        div[class *=bar ] {
                            background-color: red;
                        }
                        Selecciona los elementos que tengan un atributo con el nombre class y al menos uno de los valores del atributo llamado class es bar.

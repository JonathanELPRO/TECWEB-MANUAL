/* p,div{ color:red; } */ /* selecciona a dos elementos los agrupamos y les damos los mismos estilos se puede hacer lo mismo con clases */ /* .uno,.dos{ color:red; } */ /* .contenedor p{ background-color: orange; } */ /* quiero seleccionar a todos los elementos p que son descendientes de un elemento dado por el selector ".contenedor"*/ Selectores

Seleccionan parte de un elemento en lugar del elemento mismo. Los pseudo-elementos se añaden a los selectores y permiten añadir estilos a una parte concreta de los elementos seleccionados. Por ejemplo, el pseudo elemento ::first-line selecciona solo la primera línea del elemento especificado por el selector.\ Sintaxis: selector::pseudo-elemento { … }

Selectores
Selector Descripcion Ejemplo de uso
LISTA DE SELECTORES 
                        Selector-1, Selector-2, … {   }
                        Selecciona la unión de los elementos seleccionados por cada selector de la lista
                    
                        p,div{
                            color:red;
                        }   
                        Para este ejemplo se usaron dos selectores los dos selectores de tipo la regla o estilo afectara a los los elementos que engloben dichos selectores
SELECCIONAR ELEMENTOS DESCENDIENTES (no necesariamente hijos hablamos de toda la descendencia) 
                        Sintaxis 
                        A B {}
                        Donde A y B son selectores 
                        Seleccionamos todos los elementos seleccionados por B que son descendientes de los elementos seleccionados por A
                    
                        .contenedor p{
                            background-color: orange;
                        }
                        Seleccionamos todos los elementos <p> que son descendientes de los elementos que selecciona .contenedor que puede ser solo uno.
SELECCIONAR ELEMENTOS HIJOS 
                        Sintaxis A>B{}
                        Donde A y B son selectores 
                        Seleccionamos todos los elementos seleccionados por B que son hijos de los elementos seleccionados por  A
                    
                        .contenedor>p{
                            background-color: orange;
                        }
                        Estamos seleccionando todos los elementos <p> que son solo hijos de los elementos que selecciona .contenedor
